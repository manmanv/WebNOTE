| 方法             | 说明                           | 归属   |
| ---------------- | ------------------------------ | ------ |
| alert(msg)       | 浏览器弹出警示框               | 浏览器 |
| console.log(msg) | 浏览器控制台打印输出信息       | 浏览器 |
| prompt(info)     | 浏览器弹出输入框，用户可以输入 | 浏览器 |

```
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Title</title>
	<script>
        // 输入框
        prompt('请输入')
		
		// 弹出警示框 输出展示给用户的内容
		alert('耶斯')
		
		// 控制台输出 给程序员测试
        console.log('你好程序员');
	</script>
</head>
<body>
	<div></div>
</body>
</html>
```
