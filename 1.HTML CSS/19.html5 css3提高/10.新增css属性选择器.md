```
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Title</title>
<style>
input[value]{
color: #ffddda;
}
input[type="text"]{
color: #ffddda;
}
div[class^="ico"]{
color: #ffddda;
}
div[class$="ico"]{
color: #948bff;
}
div[class*="www"]{
color: #7CBCEE;
}
</style>
</head>
<body>
<!--1.利用属性选择器可以不用借助类或者id选择器-->
<input type="password" value="请输入用户名">ico
<!--2.属性选择器还可以选择属性=值的某些元素-->
<input type="text">
<!--3.属性选择器可以选择属性值开头的某些元素-->
<div class="ico1">1</div>
<div class="ico2">1</div>
<div class="ico3">1</div>
<div class="ico4">1</div>
<!--4.属性选择器可以选择属性值结尾的某些元素-->
<div class="1ico">2</div>
<div class="2ico">2</div>
<div class="3ico">2</div>
<div class="4ico">2</div>
<!--5.属性选择器可以选择包含某些内容的属性的某些元素-->
<div class="www1">3</div>
<div class="www2">3</div>
<div class="www3">3</div>
<div class="www4">3</div>
</body>
</html>
```
